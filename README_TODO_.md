1. 향후 과제 및 고도화 제안 (Next Steps)

프로젝트의 완성도를 높이기 위해 다음 작업들을 수행하는 것을 권장합니다.

🚀 고고도 전용 추론 환경 구축

고고도 전용 스크립트 작성: 고고도 학습 결과물인 best.pt를 기반으로 140m 환경에 최적화된 추론 파일을 작성해야 합니다.

추론 방식 다각화: Low Altitude와 마찬가지로 상황에 따라 Standard, SAHI, Dual-scan 등 여러 버전을 나누어 운용하는 것이 효율적입니다.

🤖 IF 투트랙 통합 추론 모델 (Integrated Pipeline)

드론 이미지의 **메타데이터(Exif 고도 정보 등)**를 추출하여, 고도에 따라 Low 전용 모델 또는 High 전용 모델을 자동으로 선택하여 실행하는 'IF 투트랙 통합 추론 스크립트' 작성을 추천합니다.

📈 성능 최적화 및 파인튜닝

파라미터 우선 조정: 탐지 성능 변화를 위해 conf, iou, imgsz 등의 파라미터를 우선적으로 튜닝하십시오.

개별 파인튜닝: 데이터의 특성이 다르므로 고도 모델(Low/High)에 따라 파인튜닝 프로세스를 각각 독립적으로 가져가는 것이 정확도 향상에 유리합니다.
